# How to Get Your Server ID

## Quick Steps:

1. **Open Discord** (desktop app or web)
2. Go to **User Settings** (gear icon ⚙️ at bottom left)
3. Click **"Advanced"** in the left sidebar
4. Enable **"Developer Mode"** (toggle it ON)
5. **Close settings** and go back to your server
6. **Right-click** on your **server name** (at the very top of the channel list, where the server icon/name is)
7. Click **"Copy Server ID"** or **"Copy ID"**
8. **Paste it here** - it will be a long number like `987654321098765432`

---

```
│ # general               │

└─────────────────────────┘
```

After right-clicking → "Copy Server ID"

---

## Alternative Method:

If right-click doesn't work:
1. Make sure Developer Mode is enabled
2. Try right-clicking on the server icon itself
3. Or right-click in an empty space near the server name
