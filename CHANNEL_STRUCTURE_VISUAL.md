## Permission Templates

### Public Channel Template:
- @everyone: Read Messages, Send Messages
- Muted role: No permissions
- Bots: All permissions

### Read-Only Template:
- @everyone: Read Messages
- Staff: Read Messages, Send Messages
- Bots: All permissions

### Staff-Only Template:
- @everyone: No permissions
- Staff: All permissions
- Bots: All permissions

### Voice Channel Template:
- @everyone: Connect, Speak, Use Voice Activity
- Muted: No permissions
- Staff: All permissions

## Channel Order Priority

1. **Information** - Most important, users see first
2. **Account & Rewards** - Important for functionality
3. **General Chat** - Most used
4. **Gameplay** - Game-specific
5. **Voice Channels** - Easy access
6. **Support** - Available but not prominent
7. **Staff** - Hidden from members

## Customization Tips

### Channel Icons:
- Use emojis in channel names for visual appeal
- Keep consistent emoji style
- Don't overdo it

### Channel Topics:
- Set helpful topics for each channel
- Include rules/guidelines in topic
- Update topics as needed

### Slow Mode:
- Enable on #general (5-10 seconds)
- Enable on #suggestions (30 seconds)
- Enable on #trading (10 seconds)
- Don't enable on #announcements

### Pinned Messages:
- Pin important info in each channel
- Keep pinned messages updated
- Don't pin too many (max 3-5 per channel)

1. **Essential Channels:**
   - #welcome

   - 1-2 voice channels

2. **Add Later:**
   - More gameplay channels as community grows
   - More voice channels as needed
   - Specialized channels based on demand

**Remember:** You can always add channels later. Start simple!
