### Step 3: Configure the Panel

Ticket Tool will ask you for:
1. **Panel Name** - e.g., "Support Panel"

### Step 4: Add Multiple Buttons

When configuring buttons, you can add multiple:

**Option 1: Using the command multiple times**
```
/panel create name:"Support Panel" channel:#ticket-panel button:"General Support" button:"Ban Appeals"
```

**Option 2: Using Ticket Tool Dashboard**
1. Go to: https://tickettool.xyz/dashboard/1449896304665821474/configs#pc-menu
2. Login with Discord
3. Go to "Panels" section
4. Click "Create Panel"
5. Add multiple buttons/options
