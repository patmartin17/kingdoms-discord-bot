## Requirements

**Server Boost Level:**
- **Level 1**: Static banner (image)
- **Level 2**: Animated banner (GIF) ‚ú®
- You have Tier 3, so you can use animated GIFs!

**Image Requirements:**
- **Size**: 960x540 pixels (16:9 aspect ratio)
- **Format**: PNG, JPG, or GIF
- **File Size**: 
  - Static images: Up to 8MB
  - Animated GIFs: Up to 8MB
- **Aspect Ratio**: Must be 16:9 (width:height)

---

## Step-by-Step Setup

### Step 1: Prepare Your Banner
1. Create or find your GIF/image
2. Resize to **960x540 pixels** (if needed)
3. Ensure file size is under 8MB
4. Save it somewhere accessible

**Free Tools to Resize:**
- Online: https://ezgif.com/resize (for GIFs)
- Online: https://www.iloveimg.com/resize-image (for images)
- Or use Photoshop, GIMP, etc.

### Step 2: Upload Banner in Discord

1. **Open Discord** ‚Üí Go to your server
2. Click **Server Name** (top left) ‚Üí **Server Settings**
3. In left sidebar, click **Server Banner**
4. Click **Upload Banner** button
5. Select your GIF/image file
6. Preview will show how it looks
7. Click **Save Changes** at the bottom

**Alternative Path:**
- Right-click server icon ‚Üí **Server Settings** ‚Üí **Server Banner**

### Step 3: Enable Banner Display

After uploading, make sure:
- **Server Boost Level** is at least Level 1 (you have Tier 3, so ‚úÖ)
- Banner will automatically appear at the top of your server

## Tips

**For Animated GIFs:**
- Keep file size reasonable (under 5MB recommended)
- Shorter loops work better
- Test the GIF before uploading

**For Static Images:**
- High quality PNG works best
- Can use JPG if file size is an issue

**Banner Visibility:**
- Shows at the top of your server for all members
- Only visible if server has at least Level 1 boost
- Can be changed anytime in Server Settings

---

## Troubleshooting

**"Upload Banner" button is grayed out:**
- Your server needs at least Level 1 boost
- Check Server Settings ‚Üí Server Boost Status

**Banner doesn't animate:**
- Make sure you uploaded a GIF file (not just renamed PNG)
- Server needs Level 2 boost for animated banners
- Check file extension is `.gif`

**Banner looks stretched/cropped:**
- Ensure your image is exactly 960x540 pixels
- Use 16:9 aspect ratio
- Discord will crop if dimensions don't match

**File too large:**
- Compress the GIF/image
- Use tools like https://ezgif.com/optimize (for GIFs)
- Reduce quality slightly if needed
